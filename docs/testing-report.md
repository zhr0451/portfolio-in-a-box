# Отчет по тестированию

Дата: 2026-03-19

## Что проверено

- `pnpm build`
- `pnpm test:e2e`
- `pnpm test`

## Результат

- Сборка Astro проходит без ошибок.
- E2E smoke-тесты проходят: главная страница открывается по базовому пути GitHub Pages, ассеты загружаются по корректному базовому пути, а переключение языка на `ru` сохраняется после перезагрузки.
- Критических багов, регрессий и падений в текущем прогоне не найдено.

## Остаточные риски

- Автотесты покрывают только smoke-сценарии, без визуальной проверки макета.
- Не проверялись реальные внешние ссылки `itch.io`, `Telegram` и `mailto` в браузере.
- Не проверялся фактический deploy в GitHub Pages, только локальная сборка и preview-окружение.
